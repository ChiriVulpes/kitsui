Creates styles that will be rendered after the given dependency styles. Useful for ensuring CSS specificity or cascading order when styles depend on others.
const base = Style.Class("base", { color: "black" });
const accent = Style.after(base).Class("accent", { color: "red" });
// In the stylesheet, .base appears before .accent

Registers global CSS reset rules that, when mounted, are are placed before all other styles in the generated stylesheet. The definition uses the * (universal) selector. Supports nested selectors such as pseudoBefore and pseudoAfter for targeting *::before and *::after.

  boxSizing: "border-box",
  margin: "0",
  padding: "0",
  ...pseudoBefore({ boxSizing: "border-box" }),
  ...pseudoAfter({ boxSizing: "border-box" }),
}).appendTo(document.head);

// * { box-sizing: border-box; margin: 0; padding: 0 }

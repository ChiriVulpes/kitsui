breakdownExtension

Breaks a source state into keyed reusable parts owned by the provided owner. Each unique key creates a component once. Later breakdown passes reuse that component, update its per-part state, and let callers reposition it through normal placement APIs. Parts omitted from a later pass are removed and their part state is disposed.
Component.Breakdown<TValue>(owner: Owner, state: State<TValue>, breakdown: BreakdownRenderer<TValue>): CleanupFunction
owner: Owner
The owner that explicitly owns every created part.
state: State<TValue>
The source state to break down on each update.
breakdown: BreakdownRenderer<TValue>
Called immediately and on each source update to register keyed parts.

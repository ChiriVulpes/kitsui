Creates a reactive state container with an initial value. When called with an owner, the state is automatically disposed when the owner is disposed. When called without an owner, the state must gain an owner before the next tick. If the ownerless state is subscribed to by a non-State owner (e.g., a Component), that owner becomes the implicit owner. If a different non-State owner later subscribes, the implicit owner is cleared and an explicit owner is required.
